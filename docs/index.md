# Pitagora Network - Events

OSSコミュニティである Pitagora Network のメインの活動 Pitagora Meetup は、特定の組織や企業などに属さない、有志による会として運営されています。アカデミアや企業などの所属、研究者・開発者・営業など職種を問わず、技術トピックに興味のある方なら誰でもご参加いただけます。

## ミートアップの過ごし方

初めて参加される方はぜひ [過ごし方ガイド](meetup/whatis) をご覧ください。
