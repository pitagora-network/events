## まとめ

### 那須野

- Ubuntu の [Workshop](https://ubuntu.com/workshop) を試している
  - LXC/LXD ベースのコンテナ隔離環境
  - workshop 定義の yaml 仕様[リファレンス](https://ubuntu.com/workshop/docs/reference/definition-files/workshop-definition/)
  - 最小限の YAML で `workshop launch` してみた
    ```yaml
       name: sandbox
       base: ubuntu@24.04
       sdks: []
    ```
  - Dockerインストール済み環境で iptables FORWORD ポリシーが DROP の場合 outbound の NAT が機能しないので注意  
     → `iptables -P FORWARD ACCEPT` した
  - 設定ファイル一発で隔離された開発環境を作るのに最適かも。
  - Debian環境でsnapとlxdを入れて始めようとすると zfs モジュールを要求されるのでやめておいたほうが無難（今後回避できるようになる？）
  - https://github.com/gonta223/humanizer-ja
